# Calculator Application (Java)

A simple **calculator application** developed in Java that performs basic arithmetic operations. The app takes two numbers and an operator as input and returns the result of the operation.

### Features

* Accepts **two numeric inputs** from the user.
* Supports basic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Displays the result of the chosen operation.
* Simple and user-friendly interface.

### Technology Used

* **Language:** Java / Kotlin
* **Platform:** Android Studio
